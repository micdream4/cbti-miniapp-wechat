# CBT-I 睡眠助手微信小程序

一个面向微信端的 CBT-I 自助记录工具，目标是把执行成本压到最低：
- 每天 1 分钟内完成睡眠日记
- 以固定起床时间为锚点，按 7 天周期生成睡眠窗口建议
- 同时支持纸质模板下载，降低对手机的依赖

## 当前功能

1. 首页：显示今晚建议执行时段、当前睡眠窗口、执行锚点、即时反馈
2. 日记页：记录上床/起床、在床清醒、离床清醒、小睡、睡前习惯、备注
3. 历史页：查看/编辑/删除日记，并按 7 条记录生成周总结
4. 指南页：介绍 CBT-I、睡眠限制原则与执行步骤
5. 模板页：复制或打开下载链接，获取纸质记录表
6. 本地存储：无需后端，用户数据默认保存在微信本地缓存

## 核心规则

### 1. 日记默认值

- 起床时间默认跟随当前执行锚点
- 上床时间默认沿用上一条记录
- 数值项默认 `0`
- 备注默认为空
- 睡前习惯默认沿用上一条记录

### 2. 睡眠计算

- 床上时长 `TIB = 上床到起床总时长 - 夜间离床清醒时长`
- 总睡眠时长 `TST = TIB - 入睡花费 - 夜间醒后在床清醒 - 早醒后在床未再睡`
- 白天小睡不计入夜间总睡眠时长，只用于习惯分析

### 3. 睡眠窗口

- 先固定起床时间
- 首周默认只记录，满 7 条记录后再给建议
- 候选睡眠窗口 = 近 7 条记录平均总睡眠时长 + 15 分钟缓冲
- 睡眠窗口限制在 `5.5 小时 - 9 小时`
- 用户点击“生效本周窗口”后才进入下一周期

### 4. 周总结

- 从最早记录开始，按固定 7 条记录分为第一周、第二周……
- 历史页按倒序显示
- 每个周总结插入在该周最后一天记录的上方
- 周总结包含：一周表格、平均睡眠、平均效率、做得好的地方、需要关注、下周重点、下周建议窗口
