# 上线检查清单

## 自动检查

1. 运行 `node release/冒烟测试脚本.js`
2. 确认输出为 `Smoke tests passed`
3. 运行全量语法检查：

```bash
find . -name '*.js' -print0 | xargs -0 -I{} node --check "{}"
```

## 真机冒烟

1. 首页
- 首周仅记录时，不显示建议时段
- 满 7 条记录后，显示候选窗口与“生效本周窗口”
- 修改锚点时间后，首页建议同步变化

2. 日记页
- 起床时间默认跟锚点一致
- 上床时间默认沿用上一条记录
- 数值项默认 `0`
- 备注默认空
- 睡前习惯沿用上一条记录
- 不填写上床/起床时间时，提交会拦截

3. 历史页
- 删除有二次确认
- 周总结出现在正确的周边界位置
- 展开周总结后，一周表格与分析完整

4. 模板页
- 百度网盘链接可复制
- “下载并打开”在网盘场景下会提示复制到浏览器/网盘 App 打开

## 发布前后台

1. `AppID` 正确
2. 隐私保护指引已发布
3. 体验版真机通过
4. 审核备注已同步到最新功能
5. 合法域名白名单已配置（如使用直链）

## 已知边界

1. 周总结按“7 条记录”切分，不按自然周切分
2. 如果中间缺一天，周总结周期会跨超过 7 个自然日
